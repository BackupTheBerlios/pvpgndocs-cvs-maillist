Sorry about the big email you just recieved, hehe

Files of most interest are:
- pvpgndocs.sgm - This is the root of the DocBook "book".
- part1.sgm, part2.sgm, part3.sgm - These are where the chapters are 
defined
- preface.sgm - the preface
- glossary.sgm - the glossary
- doc1.sgm - The only file with content as of yet.

"DocBook: the definitive guide" is the official documentation for 
DocBook.
   View online here:
   http://docbook.org/tdg/en/html/docbook.html

   or, download the HTML files in an archive here:
   http://docbook.org/tdg/en/tdg-en-html-2.0.10.zip
   note that the front page of the book is docbook.html not index.html

The PvPGN docs are using DocBook V4.3


Here is a basic structure of the docs that i have put together, please 
tell me if I've missed anything:

   part1 - Server Admin's Guide
