<sect1 id="doc1">
  <title>Introduction to PvPGN</title>
  <sect2>
    <title>What is PvPGN?</title>
    <para>
      PvPGN is software based on bnetd that emulates the Blizzard Battle.net (TM)
      server. It currently supports all Battle.net clients such as StarCraft, Diablo II
      and Warcraft III.  It gives you the power to run your own server, manage your own
      users, run your own tournaments etc.
    </para>
    <para>
      A PvPGN Server can be used:
    </para>
    <para>
      <itemizedlist mark="bullet">
        <listitem>
          <para>when wanting to host a local tournament with local users and statistics
          (lots of internet cafe's do that)</para>
        </listitem>
        <listitem>
          <para>when wanting to maintain a local community, Battle.net is too big for some
          people; having a local friends community is something better in that case</para>
        </listitem>
        <listitem>
          <para>for people who want to play LAN but with battle.net-like statistics</para>
        </listitem>
        <listitem>
          <para>for people who want more features than Battle.net (just look at PvPGN's
          command set to see how is PvPGN better than Battle.Net)</para>
        </listitem>
      </itemizedlist>
    </para>
    <para>
      It puts you as a "System Administrator" in control.  Use a PvPGN server
      to host your own LAN parties or put the server online and on the Internet for everyone.
      You can run tournaments, have your own local competition ladders. The power is in your hands.
    </para>
    <para>
      PvPGN is available as free software licensed under the <ulink url="http://www.gnu.org/copyleft/gpl.html">GPL</ulink>.
      It has been made entirely by volunteers and is provided "AS IS", without any warranty.
    </para>
    <para>
      The official PvPGN website is located at <ulink url="http://pvpgn.berlios.de/">http://pvpgn.berlios.de</ulink>
    </para>
  </sect2>
  <sect2>
    <title>Downloading PvPGN</title>
    <para>
      The official download site for PvPGN is <ulink url="http://pvpgn.berlios.de/">http://pvpgn.berlios.de</ulink>
    </para>
    <para>
      Alternative download sites:
      <itemizedlist mark="bullet">
        <listitem>
          <para><ulink url="http://pvpgn-files.ath.cx/">http://pvpgn-files.ath.cx</ulink></para>
        </listitem>
        <listitem>
          <para><ulink url="http://www.pvpgn.de/">http://www.pvpgn.de</ulink></para>
        </listitem>
      </itemizedlist>
    </para>
    <para>
      See the install guide to help figure out exactly which package you need
    </para>
  </sect2>
  <sect2>
    <title>Getting support</title>
    <para>
      If you are still having trouble after reading the documentation, you should try the
      <ulink url="http://forums.pvpgn.org/">forums</ulink>.  <emphasis>Please</emphasis>
      search the forums before asking a question, if you ask a question that has already
      been answered you will most likely get a "search the forums" response or no response at all.
    </para>
  </sect2>
  <sect2>
    <title>Guidelines when asking for support</title>
    <para>
      You will most likely get better and faster support if you provide the following:
    </para>
    <para>
      <orderedlist numeration="arabic">
        <listitem><para>Where is the PvPGN server, internal LAN or directly connected to Internet?</para></listitem>
        <listitem><para>How many players on internal LAN? (if applicable)</para></listitem>
        <listitem><para>How many Internet players? (if applicable)</para></listitem>
        <listitem><para>Details from the relevant config files (<emphasis>Don't</emphasis> paste the whole file)</para></listitem>
        <listitem><para>Relevant extracts from the bnetd.log file</para></listitem>
        <listitem><para>What PvPGN version?  What game and game version?</para></listitem>
      </orderedlist>
    </para>
  </sect2>
  <sect2>
    <title>Places of interest - PvPGN related</title>

  <sect1 id="about">
    <title>About this guide</title>
    <para>
      This guide has been created as part of a documentation project for PvPGN.  This guide is split into three main parts:
    </para>
    <para>
      <itemizedlist mark="bullet">
        <listitem>
          <para>Server admin's guide - info and HowTo's on setting up and maintaining a PvPGN server</para>
        </listitem>
        <listitem>
          <para>Gamer's guide - info and HowTo's for getting connected to and playing on PvPGN servers; and</para>
        </listitem>
        <listitem>
          <para>Developer's guide - info for wannabe PvPGN developers</para>
        </listitem>
      </itemizedlist>
    </para>
    <para>
      <emphasis>Q.</emphasis> Can you help?
      <emphasis>A.</emphasis> Yes - read this guide
    </para>
    <para>
      If you are encountering an administration issue or your clients are having operational issues then
      see if that scenario has been covered in this guide. If after doing this you have a working solution,
      then post a message to the pvpgn forums at
      <ulink url="http://forums.pvpgn.org/">http://forums.pvpgn.org</ulink>.  The documentation team (only
      2 of us at the time of writing) often browse the forums and collect information which can be used in
      the documentation.
    </para>
    <para>
      We are always looking for people to write documentation, if you are interested, please contact the
      documentation team using the feedback form on the website at
    <ulink url="http://pvpgndocs.berlios.de/feedback.php">http://pvpgndocs.berlios.de/feedback.php</ulink>
    (or if you are reading this online, the "feedback" link on the left side of the page)
