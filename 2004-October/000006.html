    <p>Before you install PvPGN, you must decide what storage mode you are going to use. "Storage" is what we call in PvPGN the method used to save user account data (usernames, passwords, win/loss records and lots more). Thus it is a very important part of PvPGN and can drastically increase or decrease speed of the server depending on the chosen method. Because we wanted it to be easy to add new storage types we designed a modular solution, something like this:</p>
!   <p align="left"><img src="http://pvpgndocs.berlios.de/storage.png" width="350" height="250"></p>
    <p align="left">PvPGN can be compiled with support for more than one storage type but (of course) at runtime it will only use one (as chosen by the server admin from the configuration file).</p>
